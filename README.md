# Project

Simple Vite + React site.

## Project info

This repository contains a Vite + React site built with TypeScript and Tailwind CSS.

## Local development

Requirements: Node.js 18+ and npm

```sh
# Clone the repo
git clone <YOUR_GIT_URL>
cd <REPO_FOLDER>

# Install deps and start dev server
npm ci
npm run dev
```

## Build & deploy

Build the static site:

```sh
npm run build
```

The site output is written to `./dist`. A GitHub Actions workflow is included to build and deploy to GitHub Pages on pushes to `main`.

## Technologies

- Vite
- React
- TypeScript
- Tailwind CSS
